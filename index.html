How to Use CypherTyper

1. Encrypting

  1. Enter ID & Message.
  2. Click **"Encrypt & Get Code"**.
  3. Copy the code from the pop-up.

2. Decrypting

  1. Switch to Decrypt Mode.
  2. Paste the **Cypher Code**.
  3. Click **"Decrypt Message"**.
  4. View decoded English message.
